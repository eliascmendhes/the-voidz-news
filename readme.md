
# Inscrição na Newsletter The Voidz

Este projeto é uma página simples de inscrição em newsletter inspirada na identidade visual da banda The Voidz. Os usuários podem inserir seus endereços de e-mail e se inscrever para receber atualizações e notícias sobre a banda.


## Recursos

-   Design responsivo
-   Campo de entrada de e-mail
-   Botão de inscrição com validação básica
-   Estilo underground com cores e fontes inspiradas em The Voidz

## Tecnologias

-   HTML5
-   CSS3
-   JavaScript
-   Google Fonts (VT323)

## Instalação e Uso

Para usar este projeto, siga estas etapas:

1.  Clone o repositório ou baixe o arquivo ZIP contendo os arquivos do projeto.
2.  Extraia os arquivos em um diretório de sua escolha.
3.  Abra o arquivo `index.html` no seu navegador web preferido.
